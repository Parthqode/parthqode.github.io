How to read this guide

Each row = one state. Apply its Surface + Text/Icon tokens exactly — that pairing passes the check. Badges: AAA ≥7 AA ≥4.5 for text · UI ≥3 for icons/borders · n/a = disabled.

Buttons — Filled primary actions

Solid surface. Because the Success/Warning surfaces are light, their label is ink (#0c0d10); the Error surface is dark enough for white. Hover/Pressed/Focus shown as static swatches, then a live button you can actually interact with.
